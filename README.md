# NotesTaker
# 📝 Roobis Notes — Smart Notes App

A modern, minimal, and real-time synced notes application built using **HTML, CSS, JavaScript, and Firebase Firestore**.
Designed for a **single user**, this app allows seamless note-taking across devices with a premium UI experience.

---

## 🚀 Features

### ✨ Core Features

* 🆕 Create, edit, and delete notes
* 🔄 Real-time sync using Firebase Firestore
* 🏷️ Add and manage tags
* 📌 Pin important notes
* 🔍 Search notes instantly
* 🔤 Sort notes (Latest, Oldest, A-Z, Z-A)

### 🧠 Smart Editing

* Rich text editor (bold, italic, lists, headings)
* Auto-save (debounced)
* Editable + locked modes

### 🎨 UI / UX

* Dark & Light theme toggle
* Smooth animations & transitions
* Clean, professional layout
* Focus mode for distraction-free writing

### 📤 Export & Utilities

* Export notes as **image (PNG)** using `html2canvas`
* Clear all notes option
* Real-time sync indicator (Saving / Saved / Error)

---

## 🏗️ Tech Stack

* **Frontend:** HTML, CSS, Vanilla JavaScript
* **Database:** Firebase Firestore
* **Libraries:**

  * Firebase CDN (compat version)
  * html2canvas (for export)
